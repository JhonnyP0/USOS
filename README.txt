TO RUN THIS APP PROPERLY YOU NEED TO MAKE .env FILE WITH PROPER DATA SUCH AS:

DB_HOST=db
DB_PORT=3306
DB_NAME=users
DB_USER=root
DB_PASSWORD= <YOUR_PASSWORD>

AS SOON AS YOU CREATE THIS FILE WITH DATA MENTIONED ABOVE, YOU SHOUD RUN [docker compose up --build]
ENVIROMENT HAS BEEN SUCCESFULY CREATED
NOW WITH HELP WITH API AND MAIN APP YOU CAN PRESENT AND CHANGE DATA FROM DATABASE
